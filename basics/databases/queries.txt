Q1 - Find all the comments for the photo with ID = 3, along with the username of the comment author ?

SELECT comments.contents, users.username FROM COMMENTS
JOIN users  ON comments.user_id = users.id
WHERE comments.photo_id = 3;

Q2 - Find the average number of comments per photo ?

SELECT AVG(total_comment) AS average_count
FROM (SELECT COUNT(*) AS total_comment
  FROM COMMENTS)

Q3 - Find the photo with most comments attached to it ?

SELECT photo_id, COUNT(contents) as comment_count FROM photos JOIN COMMENTS ON photos.id = comments.photo_id
GROUP BY photo_id
ORDER BY comment_count DESC LIMIT 1;

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


Q4 - Find the photo with ID = 10 and get the number of comments attached to it ?

SELECT count(*) FROM comments
WHERE photo_id = 10;
